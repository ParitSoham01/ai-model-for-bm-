# BlinkMoney F.I.B. — Financial Intelligence Bureau (Money MRI)

An immersive, high-fidelity single-page web application featuring a cinematic financial investigation and diagnostic scan engine. The theme balances a **70% Premium Financial Intelligence Bureau** design with a **30% Cinematic Investigation Experience** (inspired by Stripe, Spotify Wrapped, and Bloomberg HUD terminals).

## 🚀 Technology Stack
- **Structure**: Vanilla HTML5 (SEO optimized, semantic hierarchy, unique element IDs).
- **Styling**: Vanilla CSS (glassmorphic cards, glowing radial matrix connections, pulsing neon timeline pathways, dark forest-green variables).
- **Logic**: Vanilla ES6 JavaScript (compounding opportunity loss algorithms, real-time simulator timelines, and downloadable document triggers).
- **Sound Engine**: Web Audio API Sound Engine (short blips, low stamp thuds, and sweep scanner hums generated dynamically).
- **Assets**: Custom cropped detective character poses (`detective.jpg`, `detective_alt.jpg`, `detective_cutouts.jpg`) integrated into chat bubbles, report cards, and certificates.

## 📁 Key Files
- `index.html`: The consolidation of all 7 narrative stages (Onboarding, Chat, Category Leak Checklist, Connection Matrix, MRI Scanner Scanner, Scroll-driven dashboard, Timeline simulator, and Vault cabinet).
- `style.css`: Premium layout structures, glassmorphism, responsive queries, and animations.
- `app.js`: Financial computations, chat logs, sound triggers, and Canvas drawing setups for image exports.
- `detective.jpg`, `detective_alt.jpg`, `detective_cutouts.jpg`: Pinned detective assets.

## 🛠️ Verification & Running Locally
Simply spin up a local server in this directory:
```bash
python -m http.server 3000
```
Then navigate to [http://localhost:3000](http://localhost:3000).

## ⚡ Deployment
This repository is pre-configured and ready to be deployed on **Vercel** with a single click.
